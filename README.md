# OIBSIP Java Development Internship

This repository contains the tasks completed as part of the Oasis Infobyte Java Development Internship.

## Internship Domain
Java Development

## Completed Tasks

### Task 1: Number Guessing Game
A console-based Java game where the user guesses a randomly generated number within limited attempts. The project includes multiple rounds, score calculation, input validation, and final game summary.

### Task 2: ATM Interface
A console-based Java ATM simulation that allows users to login, check balance, deposit money, withdraw money, transfer money, and view transaction history. The project also includes file handling to save balance and transaction history permanently.
